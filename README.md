# 💫 About Me:

Hola! Soy Felipe, desarrollador de software con curiosidad constante por entender cómo funcionan las cosas y cómo mejorarlas.  
A lo largo del tiempo fui trabajando con distintas tecnologías en backend, frontend y cloud, y descubrí que lo que más disfruto es aprender lo que cada proyecto necesita para salir adelante.  

Mantengo un enfoque agnóstico respecto a tecnologías, priorizando siempre la solución más eficiente para cada caso.  
Lo que me motiva es resolver problemas reales, diseñar soluciones claras para entornos complejos y seguir creciendo con cada desafío.
